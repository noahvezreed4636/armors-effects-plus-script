## What It Does

Armors Effects Plus is a Minecraft mod built around armor-driven abilities. Instead of offering a broad buff pool, it links each perk to a complete matching set, so the armor you equip directly affects combat, hazards, movement, and nearby enemies.

It includes multiple material sets, such as leather, gold, chainmail, copper, iron, diamond, and netherite. Each set brings its own advantage, ranging from cleansing status effects and improving ore awareness to contact damage, lightning behavior, speed changes, and stronger defense against environmental dangers.

## Included Effects

- Effects only activate when the full matching armor set is worn
- Leather armor removes poison effects
- Gold armor prevents piglin aggression
- Chainmail armor damages nearby mobs on contact
- Copper armor blocks lightning damage
- Copper armor can trigger lightning on hit
- Iron armor increases damage output
- Iron armor slightly slows movement
- Diamond armor increases speed and detects nearby ores
- Netherite armor blocks fire, lava, and knockback

## FAQ

### How do I turn on an armor effect?
Equip the complete set for the material you want to use. The effect is bound to the full matching set.

### Will mixed armor pieces activate anything?
No. These effects are made for full sets, not mixed combinations.

### Can I switch between armor sets?
Yes. Each supported armor type offers a different gameplay advantage, so you can change sets based on the situation.

### What changes in new builds?
Updates may adjust how the armor effects work, refine interactions, or improve support for the targeted Minecraft setup.

### Where should the mod file go?
Put the mod in the Minecraft mods directory used by your installation.

### Does this alter normal Minecraft armor stats?
The mod focuses on added abilities and set-based interactions, while the armor still follows the normal game setup for each material.
